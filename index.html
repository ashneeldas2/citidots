citidots.

CitiDots visually displays bike movement and collects heat data based on area activity. The purpose of this experiment is to be able to simulate crowdedness with machine learning.

The project is based on published dock station data taken from CitiBike's database. Bikes were tracked by bike ID and used Google Maps' direction API to predict a travelled path.
